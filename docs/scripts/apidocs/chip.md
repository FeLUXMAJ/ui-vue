#### Slots

| Name      | Description                                       |
| --------- | ------------------------------------------------- |
| (default) | The default slot holds the chip inner components. |

The inner components:

- `<ui-chip-text>`
- `<ui-chip-leading-icon>`
- `<ui-chip-trailing-icon>`
- `<ui-chip-checkmark>`
