### `<ui-dialog-footer>`

#### Props

| Name       | Type     | Default    | Description             |
| ---------- | -------- | ---------- | ----------------------- |
| acceptText | `String` | `'Accept'` | The accept button text. |
| cancelText | `String` | `'Cancel'` | The cancel button text. |

#### Slots

| Name      | Description                                                     |
| --------- | --------------------------------------------------------------- |
| (default) | The default slot holds the dialog actions and can contain HTML. |
