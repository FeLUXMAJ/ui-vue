## `<ui-card>`

### Props

| Name     | Type      | Default | Description                                                |
| -------- | --------- | ------- | ---------------------------------------------------------- |
| outlined | `Boolean` | `false` | Removes the shadow and displays a hairline outline instead |

### Slots

| Name      | Description                     |
| --------- | ------------------------------- |
| (default) | Mandatory, for the card element |
