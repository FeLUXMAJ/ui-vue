## Quick Start

### 0. Requirement

* [BalmJS](https://balmjs.com/)
* [Vue.js](https://vuejs.org/)@2.1.0+

### 1. Install

```sh
npm install --save balm-ui
// or
yarn add balm-ui
```

### 2. CSS Config

```css
/* `/path/to/_vendor.scss` */

/* Sass code  */
@import 'node_modules/balm-ui/src/styles/balm-ui.scss';
```

### 3. Usage

```js
// `/path/to/your_main.js`

import Vue from 'vue';
import BalmUI from 'balm-ui';

Vue.use(BalmUI);
```
