```html
<ui-tab-bar-scroller><!-- the tab components --></ui-tab-bar-scroller>
```

#### Props

| Name    | Type     | Default | Description                                 |
| ------- | -------- | ------- | ------------------------------------------- |
| scrollX | `Number` | `0`     | The current scroll value.                   |
| align   | `String` | `null`  | Tab alignment. `['start', 'center', 'end']` |
